# 环境搭建

```bash
# Install the Angular CLI
npm install -g @angular/cli

# Create a workspace and initial application
ng new my-ng-app

# Run the application
cd my-ng-app

# launches the dev server, watches your files, and rebuilds the app as you make changes to those files.
# --open option automatically opens your browser to http://localhost:4200/
ng serve --open
# or
ng serve -o
```
